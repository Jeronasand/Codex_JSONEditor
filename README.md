# JSON Editor

一个基于 React + Vite + TypeScript 的静态 JSON Editor 项目。

在线访问地址：[https://jsoneditor.jeronasand.cn](https://jsoneditor.jeronasand.cn)

## 功能

- 文本编辑 JSON
- 校验 JSON 合法性
- 格式化 JSON
- 压缩 JSON
- 复制到剪贴板
- 上传本地 JSON 文件
- 下载当前内容

## 本地运行

确保本机已安装 Node.js 18+，然后执行：

```bash
npm install
npm run dev
```

## 打包

```bash
npm run build
npm run preview
```

构建产物会输出到 `dist/`，可以直接部署到任意静态托管服务。
